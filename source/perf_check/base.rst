3 总结
----------

#. 用户CPU和Nice CPU高, 说明用户态进程占用了较多的CPU
#. 系统CPU高, 说明内核占用了较多CPU, 应该查看内核线程或者系统调用的性能问题
#. I/O等待CPU高, 说明等待I/O的时间标记长
#. 软中断和硬中断高
